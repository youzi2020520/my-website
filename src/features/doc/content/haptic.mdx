## 特性

- 在 Android 上使用 [Vibration API](https://developer.mozilla.org/docs/Web/API/Vibration_API)，在 iOS 上使用 checkbox 技巧。
- 支持在 Android 设备上自定义持续时间和振动模式。
- 触感功能灵感来自 [Vercel](https://vercel.com) 的移动端体验。

## 用法

```tsx
import { haptic } from "@/lib/haptic"
```

```tsx
<Button onClick={() => haptic()}>Haptic</Button>
```

<Callout>
  触感反馈不应随意使用。关于最佳实践，请参阅 Apple 人机界面指南中的[播放触感](https://developer.apple.com/design/human-interface-guidelines/playing-haptics)。
