## 技术栈

- Next.js 16
- Tailwind CSS v4
- shadcn/ui

## 特性

- 简洁现代的设计
- 浅色/深色主题
- vCard 集成
- SEO 优化（[JSON-LD schema](https://json-ld.org)、sitemap、robots）
- AI 准备就绪 [/llms.txt](https://llmstxt.org)
- 防垃圾邮件保护
- 可安装为 PWA
- 使用 [PostHog](https://posthog.com) 和 [OpenPanel](https://openpanel.dev) 进行分析
- 通过 [c15t](https://c15t.com) 进行 Consent 管理

## 内容

由 MDX 驱动的集中文档系统：

- 博客文章和组件文档的统一内容层
- 基于类别的内容组织
- 用于 AI 可读性的原始 `.mdx` 端点
- 带代码块的语法高亮
- 用于丰富链接预览的动态 OG 图片
- 用于内容分发的 RSS 订阅源

## 注册表

使用由 [shadcn CLI](https://ui.shadcn.com/docs/cli) 提供支持的自定义注册表，轻松构建和分发可重用的组件、钩子和页面。

每个条目都有详细文档，包括：

- 实时预览和代码片段
- 美观易读的代码块
- 一键命令块（pnpm、npm、yarn、bun）
